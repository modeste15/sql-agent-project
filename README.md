#  SQL Agent with Mistral

##  Description

This project enables automatic SQL query generation from natural language using a Mistral-based AI agent.

The goal is to simplify database interactions by allowing users to ask questions in plain English and receive executable SQL queries.

---

## Dataset

The project uses the Uber SQL dataset available on Kaggle:
[https://www.kaggle.com/datasets/rockyt07/uber-sql-database](https://www.kaggle.com/datasets/rockyt07/uber-sql-database)

Tables included:

* Cancellations
* users
* trips
* locations
* payments
* reviews
* riders
* drivers 



##  Technologies

* Python (uv )
* Mistral AI
* SQL (PostgreSQL )
* Docker
